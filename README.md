# ROAST

**The AI-powered party game that turns your friends into debaters — and sometimes replaces them.**

---

## What is ROAST?

ROAST is a browser-based party game platform. One player hosts a room on a TV or laptop, and everyone else joins on their phones with a simple 4-letter code. No downloads, no accounts, no friction — just pull up the site and play.

## The Game: Hot Take

Hot Take is a fast-paced debate game for 3-8 players. Here's how it works:

1. **AI generates a spicy prompt** — something like *"Pineapple on pizza is the greatest culinary innovation of the 21st century."*
2. **Two players are chosen to debate** — one argues FOR, one argues AGAINST.
3. **Everyone else votes** on who made the better argument.
4. **The twist**: in some rounds, one debater is secretly fed an AI-generated argument. Voters earn bonus points for sniffing out which debater had robot help.

Five rounds. ~90 seconds each. A full game wraps up in about 10 minutes.

## Key Features

- **Real-time multiplayer** — instant sync between the host screen and every player's phone, powered by WebSocket rooms.
- **AI-generated prompts** — every debate topic is freshly generated by Claude, so no two games feel the same.
- **AI-detection minigame** — adds a layer of strategy: are you hearing a real opinion, or a machine-crafted argument?
- **Zero setup for players** — join with a room code from any mobile browser. No app install required.

| Layer | Technology |
|-------|------------|
| Framework | Next.js (App Router) |

| AI | Anthropic Claude API |

## Getting Started

```bash
git clone <repo-url> && cd roast
npm install
cp .env.example .env.local   # add your ANTHROPIC_API_KEY
npm run dev                   # app on :3000, PartyKit on :1999
```
